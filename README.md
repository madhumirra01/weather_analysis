# weather_analysis
This project analyzes daily weather data using Python.
It performs statistical analysis and visualization to understand temperature trends, rainfall patterns, and weather conditions.

Tools & Libraries Used:
Python
Pandas
Matplotlib

Files in this project:
weather_data.csv → Dataset containing date, temperature, and rainfall
weather_analysis.py → Python script for analysis and visualization
README.md → Project documentation

Features Implemented:
Average temperature calculation
Hottest and coldest day identification
Rainy day count
Line plot for temperature trends
Bar chart for rainfall
Histogram for temperature distribution
Correlation analysis between temperature and rainfall
Feature engineering to classify weather type

Sample Visualizations:
Temperature trend over days
Rainfall distribution
Temperature histogram
Average temperature reference line
